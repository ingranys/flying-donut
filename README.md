## Getting Started

Get a local copy up and run main script.

1. Clone the repo
   ```sh
   git clone https://github.com/ingranys/flying-donut.git
   ```
2. Make sure python3 [:link:](https://www.python.org/downloads/) is installed
   ```sh
   python3 --version
   ```
3. **Run script**.
   ```sh
   python3 donut.py
   ```
